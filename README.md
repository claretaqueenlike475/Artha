## 🧭 What Artha does

Artha is an AI agent for financial analysis. It helps you fetch data from news, the web, and the stock market. You can also ask it to do technical analysis and fundamental analysis in plain language.

Use it to:
- check market news
- review stock data
- compare companies
- run technical charts
- run basic financial checks
- get a short decision summary from the agent

Artha is built for users who want answers without digging through many sites and reports.

## 🛠️ Common Windows issues

### App does not open
- Check that you extracted the ZIP file first
- Make sure you double-click the correct file
- Try running it again as an administrator
- Restart your computer and try once more

### Browser page does not load
- Check your internet connection
- Refresh the page
- Try another browser
- Clear the browser cache if the page still fails

### Data does not appear
- Check that you entered the right query
- Make sure any API key or setup step is complete
- Try a simpler request first
- Use a different stock name or ticker symbol

### Security prompt appears
- Windows may ask for permission before opening the app
- Choose Yes if you downloaded Artha from the link above
- If the file looks changed or broken, download it again
